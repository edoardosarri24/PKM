Il [cmake](cmake.md) permette di definire macro e functioni.
I valori delle funzioni sono solo stringhe. Per lavorare con booleani (i.e., interi) si hanno delle convenzioni, come True o Ok che sono valori booleani veri.
# Funzioni
Le più usate, come ci dice la documentazione, dono:
- set
	Imposta una variabile a un valore.
- foreach
- function
