È formata da:
- JDK
	È il Java developer Kit, cioè l’insieme degli struemnti necessari per scrivere codice Java. Comprende il compilatreo javac, il javadoc, il jar, le API, l’interprete e il JRE.
- JRE
	È il Java RunTime Environment , cioè l’almbeido dove avviene l’effettiva esecuzione di programmi Java. Comprende la JVM e le librerie Java.
- JAR
	È il formato che permette in modo compatto di esportare librerie java. È assimilabile ad un packege.
- API
	Le API fornite con il JDK esistono con tre livelli di specificità:
	- SE
		È lo Standand Editiion. Comprende le librerie necessarie per lo svluppo di applicazioni desktop.
	- EE
		È la versione Enterprise edition. Continiene le librerie della versione SE più librerie per lo sviluppo di applicaioni client-server.
	- ME
		È la versione Micro edition. Continiene le librerie della versione EE più librerie per lo sviluppo di applicaioni che girano su dispositivi con capacità ridotte (Dispositivi portatili).
