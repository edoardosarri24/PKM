# quarkus
una specie di application server di microservizi in modo più leggero e facile che semplifica il deployment. stessi pricnipi di JEE ma per i microservizi: non solo http, asincrono, db ridotti.
dal grafo delle dipendenze si hanno delle metriche (es: consumo di risorse).
le specifiche oltre a quelle di JEE si chiama java profile e si possono estrare metriche dai microservizi si possono estrarre delle metriche e tramite jequer si possono estrarre delle metrcihe.
##### applicazioni
ci sono già implementazioni in quarkus che si possono usare e non fare da zero. fare qualcosa da zero mi serve per capire.
