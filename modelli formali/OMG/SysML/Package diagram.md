Definiscono name space per organizzare la struttura e la gerarchia dei modelli usati all'interno di un sistema.
Il modello può essere organizzato in molti modi, tra cui:
- Gerarchico
	Può essere a livello di azienda, del sistema o del componente del sistema.
- Tipi di diagrammi
	Comportamentali, di requsisti o di use case.
